# Rhys Cutforth CV

Static CV website ready for GitHub Pages.

## Quick Start

1. Edit content in `index.html`.
2. Commit and push to `main`.
3. In GitHub, open Settings -> Pages.
4. Set Source to GitHub Actions.
5. The workflow in `.github/workflows/deploy-pages.yml` will publish the site.

After the first successful run, your site will be available at:

`https://<your-github-username>.github.io/rhys/`

## Files

- `index.html`: CV content and page structure.
- `styles.css`: Visual design and responsive layout.
- `.github/workflows/deploy-pages.yml`: Automatic GitHub Pages deploy.
